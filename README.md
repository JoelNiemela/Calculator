## Features

### Variables
Supports assigning custom variables:
```ruby
x : 40
y : 2
x + y
42
```

### Unicode support
```ruby
√(π)
1.7724538509055743
```

### Predefined mathematical constants
```ruby
e : 2.71828182846
π : 3.14159265359
```

### Predefined functions
```ruby
sin
cos
tan
ln
log
sqrt
cbrt
```
#### Unicode aliases
```ruby
√ : sqrt
∛ : cbrt
```
